Career Pathway - Gobi Arts & Science College
A comprehensive web application developed for the PG & Research Department of Computer Science at Gobi Arts & Science College (GASC). This platform is designed to streamline student progress tracking, testing, and staff-managed career development.

🌐 Live Demo

🔗 **Website:** https://careerpathway.rf.gd

🚀 Overview
The Career Pathway system serves as a bridge between academic learning and professional readiness. It provides a structured environment for students to undergo regular assessments and for staff to verify progress through a centralized dashboard.

✨ Key Features
Comprehensive Dashboards: Tailored views for Students, Staff, and Administrators.

Tiered Testing System:

Daily Tests: For consistent skill reinforcement.

Weekly Assessments: To track short-term progress.

Monthly Evaluations: Comprehensive exams to gauge long-term retention.

Staff Verification: Secure workflow for staff members to validate student data and test results.

Administrative Control: Full management of user roles and system configurations.

🛠️ Technical Stack
Backend: PHP / .NET / Python (as per project implementation)

Frontend: HTML5, CSS3, JavaScript

Mobile Integration: Flutter (for mobile-responsive modules)

Database: Structured SQL for secure data management

⚙️ Installation
Clone the repository:

Configure your local server (XAMPP/WAMP or similar).

Import the database schema provided in the /config folder.

Update config/config.php with your local credentials.

Access the application via localhost/career-pathway.
